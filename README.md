# cloudforge

## Setup

### Prerequisites

- Node.js (v20 or higher)
- npm (comes with Node.js)
- Docker (Docker Desktop for Mac)

### Install

```bash
nvm install
npm install -g @nestjs/cli
```

### Local Database

```bash
docker compose up -d
```

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd frontend
npm install
```

### Run

```bash
cd backend
npm run start
```

```bash
cd frontend
npm run dev
```
